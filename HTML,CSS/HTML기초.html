목록

순서가 없는 목록 - Unordered list

  • ul은 앞에 점이 생긴다.
  • CSS로 점을 없앨 수 있다.

순서가 있는 목록 - Ordered list

  1. ol은 앞에 숫자가 생긴다.
  2. ul,ol 둘다 안에 li태그를 써서 작성해야 함.
  3. ol태그에 Type을 사용하여 숫자를 로마나 알파벳 등으로 변형할 수 있다.
  4. ol태그에 Start 속성으로 시작 번호를 설정할 수 있다.
  5. value 속성을 사용하여 임의의 숫자를 지정할 수 있다.
